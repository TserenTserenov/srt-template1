# Список venues в Энсхеде для аутрича

## Назначение документа

Список конкретных мест в Энсхеде, к которым нужно сходить для установления партнерских отношений для съемок контента и продвижения ORA.

## Целевая аудитория

Студенты University of Twente — основная аудитория, которая должна увидеть наш контент и узнать о местах Энсхеде через наши публикации.

## Цели аутрича

1. **Подтверждение участия** — venue owner готов быть представлен в приложении
2. **Коллаборационные публикации** — совместный контент в соцсетях с venue
3. **Доступ для съемок** — venue предоставляет пространство для авторского контента (не только коллабы)

## Критерии отбора venues

1. **Популярность у студентов** — место должно быть узнаваемым для аудитории Twente
2. **Визуальная привлекательность** — фотогеничный интерьер/экстерьер для контента
3. **Открытость к коллаборациям** — готовность работать с молодыми проектами
4. **Разнообразие** — покрываем разные lifestyle-категории
5. **Географическая доступность** — центр Энсхеде + кампус Twente

## Формат взаимодействия

- **Первый контакт** — личный визит (использовать venue-outreach-script.md)
- **Встреча** — структурированное обсуждение (использовать venue-partnership-meeting.md)
- **Follow-up** — подтверждение договоренностей, планирование первой съемки
